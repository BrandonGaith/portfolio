California Speed App

A California Speed Game App.
This application runs a game called "Speed" specifically a variant of speed called California Speed.
The trick to this one was using "Web Sockets" as they allow the user to play one game on two separate browsers.
Also, the added functionality to get the IP address of whoever is running the game and it allows another person on a different computer to play with them all in real time.

Hangman App

An App to play the game Hangman.
This app was designed to simply allow someone to play the classic game "Hangman"
Basic hangman rules, you start with blanks equal to the number of letters in the word and no strikes. But as you make more and more guesses correct ones fill in the blanks and incorrect ones add to the hangman.
The head, the body, two arms and two legs makes Hangman which means you lost the game!
